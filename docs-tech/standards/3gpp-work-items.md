## Overview

3GPP groups its specification work into Releases (Rel-15, Rel-16, and so on), and each Release is made up of Work Items, the individual pieces of technical work that produce or update specifications. The TR 21.9xx series below are the official per-Release summary reports, each describing the Work Items completed in that Release and their main achievements. This page links to those summaries so you can see, at Release level, what 3GPP delivered.

This is a list of technical reports summarizing 3GPP Work Items and main achievements.

The Status / timeframe column below is indicative only.

:::caution
Release status and dates are indicative. Confirm the current status of each Release against the 3GPP work plan before relying on it.
:::
