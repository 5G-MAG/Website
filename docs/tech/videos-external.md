## Overview

This page collects videos produced outside 5G-MAG that are useful background for 5G media and broadcast topics. Unlike the [Video Library](./videos), which holds recordings of 5G-MAG workshops and calls, the talks here are hosted or presented by third parties. More entries will be added over time.

## eXtended Reality and MPEG-I Scene Description

### glTF 2.0 Extensions in MPEG and 3GPP - Real time exchange for 3D Experiences

An external talk on the GL Transmission Format (glTF) 2.0 extensions defined in MPEG and 3GPP for real-time exchange of 3D experiences. Hosted on YouTube.
