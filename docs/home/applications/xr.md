## Overview

This application area covers eXtended Reality (XR), the umbrella term for virtual, augmented and mixed reality media, delivered over 5G. It brings together a Unity player for MPEG-I Scene Description (the MPEG format that describes how 3D objects and media make up an immersive scene) and real-time avatar streaming based on the MPEG Avatar Representation Format (ARF). It is aimed at developers building immersive experiences that combine scene description, 3D media and avatar communication over a 5G network. An Application is a curated combination of existing Reference Tools assembled into a working end-to-end service, so the cards below open the underlying tool documentation. For acronyms used here, see the [Glossary](/tech/glossary).
