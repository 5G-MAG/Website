## About

OSCAR covers applications, core networks, RAN (radio access network) and related areas across five open-source projects. Sessions span 5G Core and RAN implementations, Operations and Maintenance (OAM), and applications, closing with open discussion and Q&amp;A.

## 5G-MAG's presentation

Daniel Silhavy and Nils Duval presented "5G-MAG Reference Tools" — how 5G-MAG translates specifications into reference tools for validation, verification, demonstrations and trials, and how it bridges media applications, service aspects, and connectivity enablers and features.

**When:** 12 June, 15:00 CEST, held virtually via Zoom. A recording is available.

Q&amp;A covered Non-Terrestrial Networks (NTN), smartphone interoperability, scheduling algorithms, and roaming testing.
