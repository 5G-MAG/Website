## Origins

OSMART launched in May 2022, bringing together five founding organizations: the Consumer Technology Association WAVE Project, DASH Industry Forum, DVB Project, 5G-MAG, and HbbTV Association. A second edition in 2023 added MPEG and SVTA. 5G-MAG is a co-organizer and founding member, and hosts repositories for the collaboration on its GitHub account.

## Output

The workshops produced the **OSMART Community** — a shared GitHub repository serving as the central point for open-source media software projects and shortcuts to get involved.
