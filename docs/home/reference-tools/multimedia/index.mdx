<div class="topic-lead">
Reference tools for multimedia content delivery protocols, covering DASH (Dynamic Adaptive Streaming over HTTP), HLS (HTTP Live Streaming), and CMAF (Common Media Application Format).
</div>
<div style={{margin: '8px 0'}}><a class="button button--outline button--primary" href={useBaseUrl('/reference-tools/multimedia/scope')} style={{margin: '2px 4px 2px 0'}}>Scope</a> <a class="button button--outline button--primary" href={useBaseUrl('/reference-tools/multimedia/resources')} style={{margin: '2px 4px 2px 0'}}>Resources</a> <a class="button button--outline button--primary" href={useBaseUrl('/reference-tools/multimedia/tutorials')} style={{margin: '2px 4px 2px 0'}}>Tutorials</a> <a class="button button--outline button--primary" href={useBaseUrl('/reference-tools/multimedia/tutorials#developer-exchange')} style={{margin: '2px 4px 2px 0'}}>Developer Exchange</a></div>

These tools address the application-layer delivery protocols that sit above the 5G transport, ensuring interoperability across content formats, packaging standards, and player implementations. The project is designed to support any delivery protocol; it currently implements FLUTE and ROUTE, used respectively for file and real-time object delivery over 5G Broadcast and 5G Multicast Broadcast Services (MBS). Useful for developers validating DASH/HLS players, packagers, and Content Delivery Network (CDN) integrations in 5G environments.

**Implemented specifications:** ISO/IEC 23009-1 (MPEG-DASH), IETF RFC 8216 (HLS), ISO/IEC 23000-19 (CMAF), 3GPP [TS 26.346](https://www.3gpp.org/dynareport/26346.htm) (MBMS protocols and file delivery formats).

:::tip[Where to start]
- **Scope** for the FLUTE and ROUTE architectures and specifications behind delivering DASH, HLS and CMAF content in 5G Broadcast and 5G MBS — see [Scope](./scope).
- **Resources** for the source code, repositories and latest releases — see [Resources](./resources).
- **Tutorials** for the walkthrough on delivering DASH/CMAF content over FLUTE (rt-libflute) and ROUTE (rt-mbms-mw, route-gpac branch), plus Developer Exchange — see [Tutorials](./tutorials).
:::
