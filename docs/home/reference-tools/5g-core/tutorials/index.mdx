These tutorials show how to exercise the 5GC Service Consumer libraries and their command-line tools. New here? Start with **libscPCF**, the most complete worked example (it also exercises libscBSF). The dedicated **libscBSF** tutorial is still in progress.

- [Using the 5GC Service Consumer libraries: libscPCF](./using-libscPCF): test the Policy Control Function (PCF) service consumer library (recommended starting point).
- [Using the 5GC Service Consumer libraries: libscMBSMF](./using-libscMBSMF): test the Multicast/Broadcast Session Management Function (MB-SMF) service consumer library.
- [Using the 5GC Service Consumer libraries: libscBSF](./using-libscBSF): the Binding Support Function (BSF) service consumer library (dedicated tutorial in progress; libscPCF already demonstrates it).
