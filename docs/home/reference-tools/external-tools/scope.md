This page explains what belongs on the External Tools list and why this section works differently from the other Reference Tools sections.

## What belongs here

Every other Reference Tools section covers a project built and maintained by 5G-MAG in its own GitHub repositories. External Tools is different: it lists third-party tools, built and maintained outside 5G-MAG, that can enhance the functionality and usability of the Reference Tools, help with testing and evaluation, or provide additional features for multimedia content delivery. See the [index page](./) for the current entries.

## Current entries

- **DVB NIP Analyzer** - analyzes and tests DVB-NIP streams, inspecting FLUTE sessions and the structure of the transmitted data. Related to [Multimedia Delivery Protocols](/reference-tools/multimedia). Distributed as a Docker image; see [Resources](./resources) for where to get it.

## Why there is no roadmap, repository table, or Community Stats here

Because these tools live in third-party repositories rather than a 5G-MAG one, the roadmap, repository table, and release feed that the [Resources](./resources) page carries for the other Reference Tools sections, and the GitHub activity tracking on their Community Stats sections, do not apply here. See [Resources](./resources) for where each tool's own source lives instead.
