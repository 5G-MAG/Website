DVB-I defines a common service discovery and selection framework that unifies IP-delivered TV services across access networks. The 5G-MAG implementation covers DVB-I service discovery lists, bootstrap procedures, and integration with both 5G unicast (via 5G Media Streaming, 5GMS) and 5G Broadcast (via Further evolved Multimedia Broadcast Multicast Service, FeMBMS) delivery paths, enabling a seamless hybrid broadcast/broadband TV experience.

**Implemented specifications:** DVB BlueBook A177 (DVB-I specification), ETSI TS 103 720 (5G Broadcast delivery path), 3GPP [TS 26.512](https://www.3gpp.org/dynareport/26512.htm) (5GMS unicast delivery path).
