<div class="topic-lead">
Common Tools are shared helper repositories (scripts, example configurations and build utilities) that support several Reference Tools rather than implementing a specification of their own.
</div>
<div style={{margin: '8px 0'}}><a class="button button--outline button--primary" href="/reference-tools/common-tools/scope" style={{margin: '2px 4px 2px 0'}}>Scope</a> <a class="button button--outline button--primary" href="/reference-tools/common-tools/resources" style={{margin: '2px 4px 2px 0'}}>Resources</a></div>

You normally arrive here from a specific tool's setup steps, not by starting here. See the [Scope](./scope) page for what each helper tool supports, and [Resources](./resources) for the repository and latest releases.

:::tip[Where to start]
- **Scope** for which shared helper tool in rt-common-shared supports which Reference Tool project — see [Scope](./scope).
- **Resources** for the source code, repositories and latest releases — see [Resources](./resources).

There is no dedicated Tutorials page for Common Tools: these scripts and configs are consumed as steps inside each supported project's own tutorial, not run standalone — see [Scope](./scope) for which project uses which helper.
:::
