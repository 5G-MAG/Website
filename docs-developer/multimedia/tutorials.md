Written step-by-step tutorials for the multimedia content delivery protocol tools are not yet published here. In the meantime, the videos below and the project repositories are the best starting points. For setup instructions, refer to the README in each [multimedia delivery repository](https://github.com/5G-MAG). See the [Scope](./scope) page for the specifications and architecture, and the [Software Repositories](./repositories) page for the code.
