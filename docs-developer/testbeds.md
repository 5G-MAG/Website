<h1>Testbeds and Evaluation Tools</h1>
</div>
</div>

Testbeds provide shared infrastructure and evaluation frameworks for testing 5G media technologies in realistic conditions. They complement the [Reference Tools](./projects) by offering environments where implementations can be validated end-to-end, and they underpin several of the [Applications](./applications/) scenarios.

:::note
Access arrangements differ per testbed (some are open, others are available on request or through the community channels). Confirm the current arrangement with the 5G-MAG community before relying on access; see the [Developer Community](./community) page for how to get in touch.
:::
