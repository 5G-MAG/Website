## Overview

This page lists the prebuilt GitHub packages published for the 5G Media Streaming project. These are Android library artifacts that client-side applications (such as the 5GMSd-Aware Application) depend on, so you do not have to build the libraries from source. This same list also appears on the [Repositories](./repositories) page.

This project also provides or makes use of the following GitHub Container packages.

## 5G Media Streaming
