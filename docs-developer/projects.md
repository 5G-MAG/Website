<h1>Reference Tools</h1>
</div>
</div>

Reference Tools are open-source software implementations produced by the 5G-MAG community. Each tool corresponds to a specific 3GPP or industry specification and includes documentation, source code, and tutorials. Select a tool below to explore its scope, roadmap, and getting-started guides.

Start here if you know which specification or tool you want (tool-first). If you would rather see a complete end-to-end service scenario built from several tools, see [Applications](./applications/) (scenario-first). For shared test infrastructure and evaluation frameworks, see [Testbeds and Evaluation Tools](./testbeds). Each card links only to the resources currently available for that tool, so button sets differ from one card to another.
