## Overview

Network APIs let an application talk to the mobile network through standard, portable interfaces instead of operator-specific ones. CAMARA is an open-source API project (Linux Foundation, together with the GSMA) that standardises these interfaces. This area provides CAMARA-compliant client tools so a media application can, for example, request a quality-of-service (QoS) boost for a live stream. An Application is a curated combination of existing Reference Tools assembled into a working end-to-end service, so the card below opens the underlying tool documentation.

The AI/ML Evaluation Framework and the 6G Testbed shown for this category are hosted under [Testbeds and Evaluation Tools](../testbeds); this page focuses on the CAMARA Network APIs client tooling.
